### 새 언어 추가 스크립트
```bash
# 새 ARB 파일 생성
python scripts/create_new_language.py [언어코드]

# 번역 진행률 확인
python scripts/check_translation_progress.py

# 번역 검증
python scripts/validate_translations.py [언어코드]
```
